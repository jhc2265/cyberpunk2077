# Cyberpunk 2077 — 나이트 시티 아카이브 (리디자인 콘셉트)

사이버펑크 2077을 모티프로 한 **비공식 한국어 리디자인 콘셉트** 랜딩 페이지입니다.
나이트 시티의 세계관, 도시 구역, 전설적 캐릭터들을 인터랙티브하게 탐험할 수 있도록 만든
정적(static) 웹사이트예요. 프레임워크 없이 순수 HTML / CSS / JavaScript로만 구성되어 있습니다.

> ⚠️ 팬 제작 학습용 프로젝트입니다. Cyberpunk 2077 및 관련 IP의 모든 권리는
> CD PROJEKT RED에 있으며, 본 저장소는 상업적 목적이 아닙니다.

---

## 미리보기 실행

별도 빌드 과정이 없습니다. 두 가지 방법 중 편한 쪽을 쓰면 됩니다.

**1) 가장 간단하게** — `index.html` 파일을 브라우저로 바로 열기
(일부 브라우저는 로컬 파일에서 비디오/일부 기능이 제한될 수 있습니다.)

**2) 로컬 서버 권장** — Node.js가 설치되어 있다면 포함된 미니 정적 서버 사용:

```bash
node server.mjs
```

실행 후 브라우저에서 `http://127.0.0.1:4173` 접속.
(테스트 환경: Node.js v22)

## 폴더 구조

```
.
├── index.html          # 페이지 본문(마크업)
├── css/
│   └── style.css       # 전체 스타일 · 반응형(모바일) 레이아웃
├── js/
│   └── main.js         # 인터랙션(슬라이더, 자동재생, 스크롤 등장 등)
├── server.mjs          # 로컬 미리보기용 미니 정적 서버 (배포엔 불필요)
├── assets/             # 이미지 · 영상 · 아이콘
├── .gitignore
└── README.md
```

### 주요 섹션

| 섹션 | 설명 |
| --- | --- |
| `#top` (Hero) | 메인 비주얼 영상 + 카피. 모바일에선 화면에 보일 때 자동재생 |
| `#updates` | 최신 업데이트 뉴스 카드 |
| `#characters` | 전설적 캐릭터 스포트라이트 |

| `#districts` | 도시 구역 + 인터랙티브 지도 |

## 기술 스택

- HTML5 (시맨틱 마크업)
- CSS3 — Grid / Flexbox, `aspect-ratio`, 반응형 미디어쿼리, 마스크 · 그라데이션
- Vanilla JavaScript — `IntersectionObserver` 기반 스크롤 등장 · 영상 자동재생, 슬라이더/자동전환
- 의존성 · 빌드 도구 없음

### 반응형 / 모바일 동작 메모

- 히어로 비주얼은 소재 비율(4:3)에 맞춰 잘림 없이 꽉 차게 표시됩니다.
- 모바일에서는 도시 배경이 히어로 전체에 풀블리드로 깔립니다.
- 히어로 영상은 데스크톱에서는 "지금 접속" 버튼 hover 시, 모바일/터치 기기에서는
  화면에 보이는 동안 자동재생(muted · loop · playsinline)됩니다.

## `[지워도 됨]` 표시 파일에 대하여

저장소를 가볍게 정리할 수 있도록, **코드 어디에서도 참조하지 않는 파일**들의 이름 앞에
`[지워도 됨] ` 접두어를 붙여 두었습니다. 사이트 동작에 영향이 없으므로 안심하고 삭제해도 됩니다.

## 배포 (GitHub Pages)

정적 사이트라 GitHub Pages로 바로 배포할 수 있습니다.

1. 이 저장소를 GitHub에 푸시
2. **Settings → Pages → Build and deployment**
3. Source를 **Deploy from a branch**, 브랜치를 `main` / 루트(`/`)로 지정
4. 잠시 후 발급되는 URL로 접속

배포 시 `index.html`의 Open Graph 태그(`og:url`, `og:image`)를 실제 절대 URL로 바꿔주면
소셜 공유 미리보기가 정상 표시됩니다. (`server.mjs`는 로컬 전용이라 배포에는 필요 없습니다.)
